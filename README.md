**Features:**
- [x] configurable colors, sizes, visibilities and speeds
- [x] hideable GUI
- [x] resizable canvas 
- [x] simple GUI and settings
- [x] capturing images 
- [ ] recording movies and GIFs (Comming soon)

Available settings:
- max points [float]
- update interval [float]
- spawn interval [float]
- despawn interval [float]
- move interval [float]
- line width [float]
- update [bool]
- draw lines [bool]
- draw circles [bool]
- line color [Color3]
- circle color [Color3]
- background color [Color3]
